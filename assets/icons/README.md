# 底部 TabBar 图标说明

请将以下图标资源放入此目录中（建议尺寸：81x81px，PNG格式）：

## Tab图标（未选中状态 - 灰色）
- `home.png` — 首页图标
- `map.png` — 服务/地图图标
- `plus.png` — 中间加入校园群图标（建议使用圆形深色按钮样式）
- `order.png` — 订单图标
- `profile.png` — 我的图标

## Tab图标（选中状态 - #8B6340 棕色）
- `home_active.png` — 首页图标（选中）
- `map_active.png` — 服务图标（选中）
- `order_active.png` — 订单图标（选中）
- `profile_active.png` — 我的图标（选中）

## 地图标记图标
- `marker_repair.png` — 数码维修地图标记
- `marker_trade.png` — 二手交易地图标记
- `marker_print.png` — 打印文印地图标记

## 注意事项
- TabBar 图标必须使用本地图片（不可用网络图片）
- 未选中颜色：#999999（灰色）
- 选中颜色：#8B6340（暖棕色）
- 中间"+"按钮建议设计成圆形深色样式以突显
