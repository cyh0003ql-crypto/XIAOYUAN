# Petit Aura 校园服务 - 微信小程序

## 项目简介

Petit Aura 校园服务是一款面向广州科技职业学院学生的一站式校园服务小程序，提供数码维修、二手交易、拼豆定制等校园生活服务。

## 项目结构

```
miniprogram/
├── app.js              # 小程序入口逻辑
├── app.json            # 全局配置（页面路由、TabBar）
├── app.wxss            # 全局样式
├── sitemap.json        # 搜索配置
│
├── pages/
│   ├── index/          # 首页（Petit Aura校园服务）
│   ├── map/            # 校园服务地图
│   ├── groups/         # 加入校园群
│   ├── repair/         # 数码维修
│   ├── secondhand/     # 二手交易
│   ├── orders/         # 我的订单
│   └── profile/        # 我的
│
└── assets/
    ├── icons/          # TabBar及地图图标（见icons/README.md）
    └── images/         # 页面图片资源（见images/README.md）
```

## 如何导入微信开发者工具

1. 打开 **微信开发者工具**
2. 点击 **导入项目**
3. 项目目录选择此 `miniprogram/` 文件夹
4. 填入您的 **AppID**（可使用测试号）
5. 点击 **导入** 即可

## 页面功能说明

| 页面 | 功能 |
|------|------|
| 首页 | 展示校园服务入口、成绩数据、用户评价 |
| 校园服务地图 | 地图定位服务点、分类筛选、导航 |
| 加入校园群 | 列出可加入的微信群组 |
| 数码维修 | 展示维修服务列表、在线预约 |
| 二手交易 | 回收服务、二手商品浏览、发布闲置 |
| 我的订单 | 查看历史订单状态 |
| 我的 | 个人信息、功能入口 |

## 色彩规范

| 变量 | 颜色值 | 用途 |
|------|--------|------|
| 主色 | `#8B6340` | 按钮、强调文字、选中态 |
| 背景色 | `#F5EDE0` | 页面背景、卡片 |
| 文字主色 | `#3D2B1F` | 标题、重要文字 |
| 文字副色 | `#999999` | 辅助说明文字 |
| 卡片背景 | `#FFFFFF` | 白色卡片 |
| Banner渐变 | `#C4A882 → #8B6340` | Banner区域 |

## 补充资源

在 `assets/images/` 和 `assets/icons/` 目录中放入对应图片资源后，小程序即可完整运行。  
详见各目录下的 `README.md` 文件说明。

## 开发环境

- 微信开发者工具（建议最新版）
- 微信小程序基础库 3.x
- AppID：使用测试号或正式 AppID
